机器学习基本流程

正向传播与反向传播解析

1. 机器学习基本流程

机器学习的基本流程通常包括以下几个步骤：

  • 数据收集与准备
  • 特征工程与预处理
  • 模型选择与训练
  • 模型评估与优化
  • 模型部署与监控
graph TD; A[数据收集与准备] --> B[特征工程与预处理]; B --> C[模型选择与训练]; C --> D[模型评估与优化]; D --> E[模型部署与监控];

从数据到部署，机器学习的每一步都至关重要。

2. 正向传播（Forward Propagation）

定义：计算输入数据通过网络得到输出的过程

( a^{(l)} = \sigma(W^{(l)} a^{(l-1)} + b^{(l)}) )
graph LR; X[输入 X] --> Z[线性变换 Z]; Z --> A[激活函数 A]; A --> Y[输出 Y];

代码示例


# 正向传播示例
import numpy as np

def forward_propagation(X, W, b):
    Z = np.dot(W, X) + b
    A = 1 / (1 + np.exp(-Z))  # Sigmoid激活函数
    return A

3. 反向传播（Backward Propagation）

定义：通过计算损失函数的梯度来更新模型参数的过程

( \frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} a^{(l-1)T} )
graph RL; L[损失 L] --> dA[激活梯度 dA]; dA --> dZ[线性变换梯度 dZ]; dZ --> dW[权重梯度 dW]; dZ --> db[偏置梯度 db];

代码示例


# 反向传播示例
def backward_propagation(X, Y, A, W):
    m = X.shape[1]
    dZ = A - Y
    dW = np.dot(dZ, X.T) / m
    db = np.sum(dZ) / m
    return dW, db
