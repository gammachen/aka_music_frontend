🔍 典型应用示例


### 案例：企业关系查询
**用户输入**：  
"找找腾讯投资了哪些做人工智能的创业公司"

**处理过程**：
1. 实体识别：`腾讯(公司)`, `人工智能(领域)`, `创业公司(类型)`
2. 关系路径：`腾讯 → 投资 → 公司 → 所属领域 → 人工智能`
3. 图谱查询：
```sparql
SELECT ?company WHERE {
   腾讯 :投资 ?company .
   ?company :企业类型 "创业公司" ;
            :所属领域 "人工智能" .
}
```

**返回结果**：
- 追一科技（自然语言处理）
- 优图实验室（计算机视觉）
- 微众银行（金融AI）
