Conda 是一个流行的包管理和环境管理系统，广泛用于 Python 及其相关依赖的管理。以下是一些常用的 Conda 指令：

环境管理

  1. 创建新环境

    conda create --name myenv python=3.9

    这将创建一个名为 myenv 的新环境，并安装 Python 3.9。

  2. 激活环境

    conda activate myenv

    激活名为 myenv 的环境。

  3. 列出所有环境

    conda env list

    或

    conda info --envs

    列出所有已创建的环境。

  4. 删除环境

    conda remove --name myenv --all

    删除名为 myenv 的环境。

  5. 复制环境

    conda create --name newenv --clone myenv

    复制名为 myenv 的环境并命名为 newenv。

包管理

  1. 安装包

    conda install numpy

    在当前激活的环境中安装 NumPy 包。

  2. 列出已安装的包

    conda list

    列出当前激活环境中已安装的所有包。

  3. 更新包

    conda update numpy

    更新当前激活环境中的 NumPy 包。

  4. 卸载包

    conda remove numpy

    卸载当前激活环境中的 NumPy 包。

  5. 搜索包

    conda search numpy

    搜索 NumPy 包及其可用版本。

配置管理

  1. 查看配置

    conda config --show

    查看当前的 Conda 配置。

  2. 设置配置

    conda config --set show_channel_urls yes

    设置 show_channel_urls 配置为 yes。

  3. 获取帮助

    conda --help

    或

    conda <command> --help

    获取 Conda 或特定命令的帮助信息。

清理

  1. 清理未使用的包和缓存
    conda clean --all
    清理未使用的包、缓存文件和 tarballs。

导出和导入环境

  1. 导出环境配置文件

    conda env export > environment.yml

    将当前激活的环境导出到一个 YAML 文件中。

  2. 从配置文件创建环境

    conda env create -f environment.yml

    从 YAML 文件创建环境。

这些指令是 Conda 的核心功能之一，可以帮助你高效地管理 Python 环境和依赖。
